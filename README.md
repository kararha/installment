#  نظام إدارة المبيعات بالتقسيط
# Arabic Installment Sales Management System

نظام احترافي لإدارة المبيعات بالتقسيط مبني بتقنية Flask مع واجهة مستخدم عربية حديثة.

---

##  المميزات

- ✅ إدارة العملاء (إضافة، تعديل، حذف)
- ✅ تتبع المديونيات والأقساط
- ✅ لوحة تحكم احترافية
- ✅ واجهة عربية كاملة (RTL)
- ✅ تصميم داكن احترافي
- ✅ قاعدة بيانات محلية (SQLite)
- ✅ تقارير وإحصائيات
- ✅ تطبيق سطح مكتب مستقل

---

##  طريقة التشغيل

### 1. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 2. تشغيل التطبيق

#### كتطبيق ويب (في المتصفح):
```bash
python app.py
```
ثم افتح المتصفح على: http://localhost:5000

#### كتطبيق سطح مكتب (نافذة مستقلة):
```bash
python run.py
```

---

##  إنشاء ملف تنفيذي (.exe)

### 1. تثبيت PyInstaller
```bash
pip install pyinstaller
```

> ملاحظة: تأكد أن PyInstaller يتم تشغيله من نفس بيئة Python التي ثُبتت عليها المكتبات (خصوصاً `pywebview`).
> الأفضل استخدام بيئة افتراضية (venv) ثم تشغيل PyInstaller بهذه الطريقة: `python -m PyInstaller`.

### 2. إنشاء الملف التنفيذي
```bash
python -m PyInstaller --noconsole --onefile --add-data "templates;templates" --add-data "static;static" --name "نظام_الأقساط" run.py
```

سيتم إنشاء الملف التنفيذي في مجلد `dist`

##  استخدام النظام

### إضافة عميل جديد
1. اضغط على "إضافة عميل" في الشريط الجانبي أو الزر في الجدول
2. أدخل الاسم ورقم الهاتف والمديونية الأولية
3. اضغط "إضافة العميل"

### إضافة مديونية جديدة
1. اضغط على صف العميل في الجدول
2. اختر "مديونية جديدة"
3. أدخل المبلغ والوصف

### تسديد قسط
1. اضغط على صف العميل في الجدول
2. اختر "تسديد قسط"
3. أدخل المبلغ المدفوع

### عرض التفاصيل
1. اضغط على صف العميل
2. اختر "عرض التفاصيل" أو اضغط على أيقونة العين

##  التقنيات المستخدمة

- **Backend:** Python Flask
- **Database:** SQLite + SQLAlchemy
- **Frontend:** HTML5, CSS3, JavaScript
- **UI Framework:** Bootstrap 5 RTL
- **Icons:** Bootstrap Icons
- **Font:** Cairo (Arabic)
- **Desktop:** PyWebView

##  ملاحظات

- قاعدة البيانات تُحفظ في ملف `database.db` في نفس مجلد التطبيق
- جميع البيانات تبقى على جهازك فقط
- النظام يدعم اللغة العربية بشكل كامل

###  ملاحظة حول Windows SmartScreen
عند تشغيل الملف التنفيذي لأول مرة على جهاز جديد قد يظهر تحذير **Windows protected your PC** لأن التطبيق غير موقّع (Publisher: Unknown).

- للتشغيل محلياً: اضغط **More info** ثم **Run anyway**.
- للحل النهائي عند التوزيع: استخدم **Code Signing Certificate** لتوقيع ملف الـ `.exe` (ويُفضّل EV Code Signing) حتى يقل/يختفي تحذير SmartScreen مع الوقت.
